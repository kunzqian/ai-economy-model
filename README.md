# AI 经济模型研究网站

这是关于"人工智能参与下的社会经济模型"的研究网站。

## 在线访问

（GitHub Pages 部署后添加链接）

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 构建

```bash
npm run build
```

## 项目结构

- `app/page.tsx` - 首页
- `app/about/page.tsx` - 关于页面
- `app/join/page.tsx` - 加入我们页面
- `app/thoughts/page.tsx` - 研究论文页面

## 关于

这个项目探索 AI agents 如何重塑经济结构、协作模式和价值创造机制。
